# Experimental / unvalidated

The scripts here drive the **energy-diffusion language-model** track, which is a
research sketch and is **not validated** (see the repository `STATUS.md` for the
specific known issues — e.g. the neural energy is not actually the distribution
that gets sampled, and the discrete-EDLM contrastive-divergence objective is
sign-flipped). They are kept for reference and future work, not for use.

For the validated, reproducible work, see the top-level `README.md`
("Validated: DTM / quadratic-Ising") and `scripts/dtm_ising_demo.py`.
